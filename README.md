# copilot-test-ktruchcz

A minimal Java 21 console application that prints a time-aware "Hello World" greeting to stdout.

## Quick Start

```bash
mvn verify          # build and test
mvn exec:java -Dexec.mainClass=HelloWorld   # run
```
