# copilot-test-ktruchcz

A modernized Hello World Java application built with Maven and Java 17.

## Requirements

- Java 17+
- Maven 3.6+

## Build

```bash
mvn package
```

## Run

```bash
java -jar target/hello-world-1.0.0.jar
# or with a custom name:
java -jar target/hello-world-1.0.0.jar Alice
```

## Test

```bash
mvn test
```
