# copilot-test-ktruchcz

A minimal Java Hello World application used as a baseline repository for GitHub Copilot agent experimentation.

## Quick Start

```bash
# Compile
javac HelloWorld.java

# Run
java HelloWorld
# Output: Hello World
```
