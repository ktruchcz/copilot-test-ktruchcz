# copilot-test-ktruchcz

A minimal Java "Hello World" console application used as a baseline repository for tooling experiments (e.g., GitHub Copilot).

## Overview

- **Language**: Java (no external dependencies)
- **Entry point**: `HelloWorld.java` → `public static void main(String[] args)`
- **Output**: Prints `Hello World` to stdout and exits with code `0`

## Quick Start

```bash
# Compile
javac HelloWorld.java

# Run
java HelloWorld
# Output: Hello World
```
