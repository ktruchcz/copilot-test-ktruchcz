# copilot-test-ktruchcz

A simple Java application modernized to use a standard Maven project structure with JUnit 5 tests.

## Requirements

- Java 17+
- Maven 3.6+

## Build

```bash
mvn clean package
```

## Run

```bash
java -jar target/copilot-test-ktruchcz-1.0.0.jar
```

## Test

```bash
mvn test
```
