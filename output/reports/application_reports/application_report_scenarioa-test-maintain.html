Recommendation

  • Make the Oracle 19c upgrade or migration decision before January 2027.
  • Reduce AIX dependency and evaluate Linux replatforming.
  • Use cloud migration only where it fits the remaining application lifetime.
  • Do not fund a major refactor program because the 2027 decommission plan blocks the payback case.
